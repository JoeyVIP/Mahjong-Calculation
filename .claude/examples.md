# 使用情境範例

## 使用情境範例

### 情境一：快速驗算（最常見）

使用者剛胡牌，想確認台數

1. 點開app（已經在預設的東風圈東位設定）
2. 快速點選手牌：
   - 點選「萬」tab
   - 點3次一萬、3次二萬、3次三萬
   - 點選「字」tab
   - 點3次東、2次南
3. 總共17張，結果面板自動彈出
4. 看到：自摸1台、門清1台、圈風1台、門風1台 = 4台
5. 金額顯示：每家付130元（假設50/20）
6. 完成，總耗時約15-20秒

### 情境二：學習台數

新手不確定圈風門風怎麼算

1. 輸入相同牌型
2. 看到「圈風台 1台」旁邊有說明：「東風圈時有東東東刻子」
3. 看到「門風台 1台」旁邊有說明：「東位有東東東刻子」
4. 點擊展開看到詳細規則說明
5. 理解了圈風和門風可以同時算
6. 下次就會算了

### 情境三：試算不同底台

想知道這副牌用不同底台差多少

1. 算完一次（50/20 = 130元）
2. 點擊設定，改成100/20
3. 看到金額變成180元
4. 再改成100/50
5. 看到金額變成300元
6. 了解底台比例的影響

### 情境四：複雜牌型

清一色對對胡自摸門清

1. 快速點選17張萬子（都是刻子）
2. 結果顯示：
   - 自摸1台
   - 門清1台
   - 清一色4台
   - 對對胡2台
   - 總計8台
3. 假設100/20，每家付260元

### 情境五：包含花牌

基本牌型加上春花和梅花

1. 設定為東位
2. 輸入14張牌 + 2張花（春、梅）+ 1張眼睛牌
3. 結果顯示：
   - 基本台數（自摸、門清等）
   - 花牌2台（春1+梅1）
   - 正花2台（東位的春和梅都是正花）

## 測試用牌型

### 測試一：最基本平胡
- **牌型**：一二三萬、四五六萬、七八九萬、一二三條、四四
- **設定**：自摸、門清
- **預期**：2台

### 測試二：圈風門風雙台
- **牌型**：東風圈、東位、有東東東
- **牌型**：一二三萬、東東東、五六七條、八九萬、中中
- **預期**：圈風1+門風1+其他台

### 測試三：清一色
- **牌型**：全部萬子
- **牌型**：一一一、二二二、三三三、四四四、五五
- **預期**：清一色4+對對胡2+其他

### 測試四：大三元
- **牌型**：中中中、發發發、白白白、一二三萬、四四
- **預期**：大三元8+其他

### 測試五：莊連3
- **設定**：莊家、連莊3次
- **牌型**：一二三萬、四五六條、七八九筒、東東東、南南
- **預期**：莊家台2×3+1=7+其他

### 測試六：花牌
- **設定**：東位、有春和梅
- **牌型**：基本牌型+春+梅
- **預期**：花2+正花2+其他

### 測試七：零台屁胡
- **設定**：非自摸、非門清、無任何台
- **預期**：0台，只收底

### 測試八：超高台
- **設定**：莊連5、大三元、自摸、門清
- **預期**：超過20台
