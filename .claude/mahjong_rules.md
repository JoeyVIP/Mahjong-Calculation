# 台灣十六張麻將權威規則

**依據來源**：中華民國麻將運動協會規則、麻將羅漢規則

## 胡牌結構

### 標準型：
四組三張加一對，總共十七張
- 四組可以是順子、刻子、槓子
- 一對是眼睛

### 特殊型：
七對子

## 牌的分類

- **數牌**：萬子1-9、條子1-9、筒子1-9（各4張）
- **字牌**：東南西北中發白（各4張）
- **花牌**：春夏秋冬梅蘭竹菊（各1張）

## 金額計算方式

### 公式：
**底 + 台數 × 每台金額**

例如底50元、每台20元的情況下：
- 1台 = 50 + 1×20 = 70元
- 3台 = 50 + 3×20 = 110元
- 5台 = 50 + 5×20 = 150元

### 常見的底台比例：
- **50/20**（底台比5:2，適合做大牌）
- **100/20**（底台比5:1，適合快速胡牌）
- **100/50**（底台比2:1，平衡）
- **200/50**（底台比4:1）
- **300/100**（底台比3:1，適合做大牌）

### 支付方式：
- **自摸**：三家平分，每家付的金額是「底 + 台數 × 每台金額」
- **胡別人**：放槍的人單獨付「底 + 台數 × 每台金額」

## 基本台數規則（一台）

### 自摸
自己摸到胡牌

### 門清
沒有吃碰槓，全靠自己摸牌

### 無花
手中沒有任何花牌

### 正花
花色對應座位
- **東位**：春或梅
- **南位**：夏或蘭
- **西位**：秋或菊
- **北位**：冬或竹

拿到對應的花就是正花，加1台

### 圈風刻
手中有當前風圈對應的刻子（三張）
- 東風圈有東東東 = 1台
- 南風圈有南南南 = 1台
- 西風圈有西西西 = 1台
- 北風圈有北北北 = 1台

### 門風刻
手中有自己座位對應的風牌刻子（三張）
- 東位有東東東 = 1台
- 南位有南南南 = 1台
- 西位有西西西 = 1台
- 北位有北北北 = 1台

### 圈風門風可累加：
- 東風圈的東位玩家有東東東 = 圈風1台 + 門風1台 = 2台
- 南風圈的東位玩家有東東東 = 門風1台（只有門風）
- 東風圈的南位玩家有東東東 = 圈風1台（只有圈風）

### 莊家台
莊家不論胡牌或放槍都加1台

### 莊連N台
莊家連莊N次的台數 = 2×N+1
- 上莊（第一次）= 1台
- 莊連1（連莊一次）= 3台
- 莊連2（連莊兩次）= 5台
- 莊連N = 2N+1台

## 兩台以上的項目

- **三暗刻**：2台（三組刻子都是暗的）
- **對對胡**：2台（全部都是刻子）
- **混一色**：3台（只有一種數牌+字牌）
- **碰碰胡**：4台（對對胡的別稱，但有些規則算4台）
- **清一色**：4台（只有一種數牌）
- **小三元**：4台（中發白兩刻一對）
- **四暗刻**：5台（四組刻子都是暗的）
- **大三元**：8台（中發白都是刻子）
- **小四喜**：8台（東南西北三刻一對）
- **字一色**：8台（全部都是字牌）
- **清么九**：8台（全是一和九）
- **大四喜**：16台（東南西北都是刻子）

## 花牌計算

- **每朵花**：1台
- **正花**：再加1台
- **八仙過海**：8朵花全拿，額外加8台

## 特殊胡法

- **槓上開花**：1台（開槓後摸的牌胡）
- **海底撈月**：1台（最後一張牌胡）
- **搶槓胡**：1台（別人開槓那張胡）
- **天胡**：16台（莊家起手胡）
- **地胡**：16台（閒家第一輪胡）
