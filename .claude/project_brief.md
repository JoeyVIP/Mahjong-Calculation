# 台麻計台器 - 專案簡介

## 專案名稱
台麻計台器

## 專案定位
最好用的台灣麻將計台工具

## 核心價值主張
市場上現有的麻將計台app功能雖多但操作複雜難用，我們要打造一個專注在計台核心功能、介面精美、操作流暢的現代化應用

## 專案目標
開發一個極致易用的單頁應用，讓玩家能用最快速度輸入牌型，立即獲得準確的台數計算結果

## 核心功能
- 觸控優先的大按鈕介面設計
- 快速牌型輸入流程
- 即時台數計算和視覺化呈現
- 清晰的計算說明，幫助使用者學習如何算台

## 不做的功能
- 對局記錄和歷史數據
- 多局追蹤和統計
- 玩家管理和計分板
- 這些功能會讓介面變複雜，違背我們「極致易用」的核心理念

## 目標使用者
- 需要快速驗算台數的麻將玩家
- 想學習如何算台的新手
- 重視使用體驗和介面美感的使用者

## 技術選型
- React 18搭配TypeScript
- Tailwind CSS做樣式
- Framer Motion或React Spring做流暢動畫
- 不需要後端或資料庫
- 不使用LocalStorage（避免複雜度）

## 設計原則
- **大按鈕**：所有可點擊元素至少60x60像素，適合觸控
- **快速流程**：從開始輸入到看到結果，步驟越少越好
- **視覺回饋**：每個操作都有即時的視覺和動畫回饋
- **現代美感**：使用漸層、陰影、圓角等現代設計元素
- **教育性**：顯示計算過程，幫助使用者理解台數由來

## 交付成果
- 單一HTML頁面應用
- 手機優先設計，平板和桌面等比例放大
- 可直接在瀏覽器執行
- 載入快速，動畫流暢
- 視覺精美，操作直覺
