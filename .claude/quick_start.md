# 快速開始指南

## 給Claude Code的指示

這是一個台灣麻將計台器專案，專注在「極致易用」的單一核心功能。

### 專案目標：
- 打造最好用的台麻計台工具
- 大按鈕觸控優先設計
- 精美現代的視覺介面
- 流暢的動畫體驗
- 幫助使用者學習如何算台

### 技術棧：
- React 18 + TypeScript
- Vite
- Tailwind CSS
- Framer Motion

## 專案啟動

### 1. 初始化專案
```bash
npm create vite@latest mahjong-calculator -- --template react-ts
cd mahjong-calculator
npm install
npm install -D tailwindcss postcss autoprefixer
npm install framer-motion
npx tailwindcss init -p
```

### 2. 設定Tailwind CSS
- 根據Tailwind官方文件設定
- 加入自訂顏色（麻將桌綠、金色等）

### 3. 建立專案結構
按照development_notes.md的資料夾結構

### 4. 開始開發
1. 先建立基本的Header
2. 再建立TileSelector（這是核心組件）
3. 接著HandDisplay
4. 然後SettingsPanel
5. 最後ResultPanel

### 重要提醒：
- 所有按鈕至少60x60px
- 使用Framer Motion做所有動畫
- 手機優先設計
- 保持簡潔，不要過度設計
- 專注在核心功能，不做記錄追蹤

## 關鍵設計原則

- **大**：按鈕要大，數字要大，易於觸控
- **快**：流程要快，動畫要順，立即反饋
- **美**：現代設計，漸層陰影，視覺愉悅
- **學**：幫助理解，說明清楚，降低門檻

## 第一版MVP功能清單

### 必須有的功能：
- 選擇17張牌的介面（大按鈕）
- 設定圈風、座位風、莊家
- 設定自摸/胡別人
- 設定底台金額
- 自動計算台數
- 清楚顯示結果和金額
- 流暢的動畫

### 第一版不需要的功能：
- 歷史記錄
- 對局追蹤
- 多人計分
- 統計圖表
- 匯出功能
- 任何複雜的功能

## 重點提醒

1. 這是一個手機優先的單頁應用
2. 所有按鈕至少60x60px
3. 使用現代設計風格（漸層、陰影、圓角、流暢動畫）
4. 專注在讓使用者快速輸入並獲得結果
5. 不做任何記錄追蹤功能
6. 金額計算公式：底 + 台數 × 每台金額
7. 圈風和門風可以同時計算
8. 莊家連莊公式：2×N+1台
