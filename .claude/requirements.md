# 需求規格

## 核心使用者故事

### 使用者故事一：快速選牌輸入
**身為**玩家，**我想要**用大按鈕快速點擊選擇手牌，**這樣**我在麻將桌旁也能單手快速操作

**驗收標準：**
- 所有牌面按鈕至少60x60像素
- 點擊有明確的視覺回饋（放大、顏色變化）
- 已選數量清楚顯示
- 分類切換流暢（萬條筒字花）
- 支援快速連點
- 誤觸後可輕鬆移除

### 使用者故事二：清楚的手牌顯示
**身為**玩家，**我想要**清楚看到目前選了哪些牌，**確認**沒有選錯

**驗收標準：**
- 手牌區域明顯，與選牌區視覺區隔
- 每張牌都清楚可辨識
- 可個別點擊移除
- 張數統計即時更新
- 超過17張有明顯警告

### 使用者故事三：簡單直覺的設定
**身為**玩家，**我想要**用最少的步驟完成必要設定（圈風、座位、莊家等）

**驗收標準：**
- 預設值合理（東風圈、東位、非莊家）
- 大按鈕選擇，不用下拉選單
- 設定狀態一目了然
- 常用底台比例一鍵選擇（50/20、100/20等）
- 可快速重置

### 使用者故事四：即時台數計算
**身為**玩家，**我想要**在輸入完成後立即看到台數結果，不用按計算按鈕

**驗收標準：**
- 手牌符合17張時自動計算
- 結果即時顯示
- 計算過程流暢無延遲
- 錯誤提示清楚友善

### 使用者故事五：易懂的結果呈現
**身為**玩家，**我想要**清楚看到有哪些台、總共幾台、金額多少

**驗收標準：**
- 總台數用大字體突顯
- 每個台種都列出並說明
- 圈風台、門風台特別標註
- 金額計算公式清楚顯示
- 自摸和胡別人的金額分別列出

### 使用者故事六：學習如何算台
**身為**新手玩家，**我想要**透過使用這個工具學會如何算台

**驗收標準：**
- 每個台種有簡短說明
- 為什麼符合該台種的原因清楚標示
- 可展開查看更詳細的規則說明
- 常見問題解答（如圈風門風如何算）

### 使用者故事七：流暢的動畫體驗
**身為**使用者，**我想要**每個操作都有流暢的動畫回饋，讓使用過程愉悅

**驗收標準：**
- 選牌時有放大或彈跳動畫
- 結果出現時有淡入動畫
- 數字變化有過渡動畫
- 頁面切換流暢自然
- 不卡頓，60fps

### 使用者故事八：快速重新開始
**身為**玩家，**我想要**算完一局後能快速清空重新開始下一局

**驗收標準：**
- 明顯的清空按鈕
- 一鍵清空所有手牌
- 保留設定（圈風、座位等）
- 確認提示避免誤觸

## 功能優先順序

### P0 必要功能：
- 大按鈕牌面選擇器
- 手牌顯示區
- 圈風、座位風、莊家設定
- 自摸/胡別人切換
- 底台金額設定
- 即時台數計算
- 結果展示

### P1 重要功能：
- 流暢動畫效果
- 台種說明文字
- 門清勾選
- 花牌計算
- 計算公式展示
- 清空功能

### P2 加分功能：
- 教學提示
- 規則說明彈窗
- 快捷鍵支援
- 暗牌標記
- 範例牌型快速載入

### 不做的功能：
- 歷史記錄
- 對局追蹤
- 玩家管理
- 統計分析
- 匯出功能
- 多局計分
